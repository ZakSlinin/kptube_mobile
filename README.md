<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Flutter SDK
  ```sh
  flutter doctor
  ```


### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/ZakSlinin/kptube_mobile.git
   ```
2. Install dependencies
   ```sh
   flutter pub get
   ```
